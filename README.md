# CodeAlpha_CreditScoringModel

## Overview
This project predicts whether a customer is creditworthy using Machine Learning. A Random Forest Classifier is trained on customer financial data to classify customers as Creditworthy or Not Creditworthy.

## Features
- Data preprocessing
- Feature engineering
- Random Forest Classification
- Model evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC
- New customer creditworthiness prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Google Colab

## Dataset Features
- Age
- Income
- Employment Type
- Education Level
- Marital Status
- Debt History
- Active Loans
- Savings Amount
- Overdue Payments
- Other financial attributes

## Machine Learning Model
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
